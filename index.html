Los 3 SO más usados

Microsoft | Windows 10

Según las estadísticas, continúa siendo el sistema operativo para desktop más usado, con un porcentaje, para el año 2019, del 77,83% más de 60 puntos porcentuales sobre el sistema operativo OSX.

GNU/Linux

Este es un sistema operativo libre que tiene como ventaja una mayor potencia, estabilidad, mayor seguridad ante amenazas externas y además, permite modificar el sistema de acuerdo a las necesidades del usuario.

Chrome OS

Es uno de los sistemas más recientes e innovadores, esto porque permite realizar todas las tareas desde el navegador. En ese sentido, el ordenador no requiere de un disco local grande ni de un hardware muy sofisticado.
